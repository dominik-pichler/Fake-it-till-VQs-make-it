# Deepfake Classifier Pipeline

A pipeline for AR-image attribution that classifies images into 4 categories:
- **0 - Real**: Authentic images
- **1 - LlamaGen**: Images generated by LlamaGen (B_VQ-16, L_VQ-16)
- **2 - VAR/HMAR**: Images generated by VAR/HMAR models (hmar_d20, hmar_d30, nspvar_20, nspvar_30)
- **3 - RAR**: Images generated by RAR (rar_l, rar_xxl)

## Setup

```bash
# Install dependencies with uv
uv sync
```

## Data Structure

Place your images in the following structure:

```
data/
├── train/
│   ├── real/
│   ├── llamagen_B_VQ-16/
│   ├── llamagen_L_VQ-16/
│   ├── hmar_d20/
│   ├── hmar_d30/
│   ├── nspvar_20/
│   ├── nspvar_30/
│   ├── rar_l/
│   └── rar_xxl/
├── val/
│   └── (same subdirectories as train)
└── test/
    └── (flat directory - images directly here)
```

## Usage

### Step 1: Extract Features

Extract spectral features (RGB, DCT, QFT) from all images:

```bash
uv run python deep_fake_classifier_pipeline.py extract --data-root data --out features/
```

This creates:
- `features/train_X.npy` - Training features
- `features/train_y.npy` - Training labels
- `features/val_X.npy` - Validation features
- `features/val_y.npy` - Validation labels
- `features/test_X.npy` - Test features

### Step 2: Train Model

Train classifiers (LogisticRegression, LinearSVM, HistGradientBoosting) and select the best:

```bash
uv run python deep_fake_classifier_pipeline.py train --features features/ --out models/
```

This creates:
- `models/best.joblib` - Best model based on validation accuracy
- `models/logreg.joblib`, `models/linear_svm.joblib`, `models/hgb.joblib` - Individual models
- `models/results.json` - Training metrics and comparison

### Step 3: Predict on Test Set

Generate predictions for the test set:

```bash
uv run python deep_fake_classifier_pipeline.py predict --features features/ --model models/best.joblib --out submission.csv
```

## Python API

You can also classify images directly in Python:

```python
from pathlib import Path
from deep_fake_classifier_pipeline import classify_images, CLASS_NAMES

# Classify images
img_paths = [Path("data/test/image1.png"), Path("data/test/image2.png")]
labels = classify_images(img_paths)

# Print results
for path, label in zip(img_paths, labels):
    print(f"{path}: {CLASS_NAMES[label]}")
```

### Model Path Configuration

The `classify_images` function looks for the model in this order:
1. `model_path` parameter if provided
2. `DEEPFAKE_MODEL_PATH` environment variable
3. Default: `models/best.joblib` (relative to script location)

```python
# Use custom model path
labels = classify_images(img_paths, model_path=Path("custom/model.joblib"))

# Or set environment variable
# export DEEPFAKE_MODEL_PATH=/path/to/model.joblib
```

## Feature Extraction Details

The pipeline extracts features from three "lenses":

1. **RGB Lens**: Pixel-domain statistics on high-pass residual
2. **DCT Lens**: 2D Discrete Cosine Transform statistics per channel
3. **QFT Lens**: Grayscale FFT, low-frequency band analysis

These features capture tokenizer/decoder artifacts that distinguish different generative models.
